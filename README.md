# GuessNum 猜數字（1–100）

一個純前端的 Web 猜數字遊戲，使用 HTML / CSS / JavaScript 實作，無需任何框架或後端。

## 遊戲規則

1. 系統隨機產生一個 **1~100 的整數**作為答案（每局固定不變）。
2. 玩家每次輸入猜測數字後：
   - 若猜的數字**大於答案** → 提示「**大了**」
   - 若猜的數字**小於答案** → 提示「**小了**」
3. 若猜中 → 顯示「**恭喜通過！**」，並結束本局。
4. 可按「**再玩一次**」開始新局。

## 功能

- ✅ 隨機產生 1–100 答案（每局重新產生）
- ✅ 即時回饋：大了 / 小了 / 恭喜通過！
- ✅ 本局猜測次數計數
- ✅ 動態縮小可猜範圍提示
- ✅ 局數顯示
- ✅ 最佳成績（最少猜測次數）
- ✅ 歷史紀錄列表（含時間）
- ✅ 清除紀錄功能
- ✅ LocalStorage 跨刷新保存紀錄
- ✅ Enter 鍵送出支援
- ✅ 輸入驗證（僅允許 1–100 整數）
- ✅ RWD 響應式設計

## 執行方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何依賴。
